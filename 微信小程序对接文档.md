# 微信小程序对接文档

> 本文档供微信小程序开发者阅读，说明如何对接 LOF 基金折溢价监控系统后端 API。

---

## 一、接口地址

```
后端服务地址：http://localhost:5000
```

**注意**：小程序需将 `localhost` 替换为实际服务器 IP 或域名。

---

## 二、请求格式

所有接口均使用 **HTTP GET/POST** 请求，返回格式为 **JSON**。

### 通用响应结构

```json
{
  "code": 0,
  "message": "success",
  "data": { ... }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| code | number | 0=成功，非0=失败 |
| message | string | 错误信息 |
| data | object | 返回数据 |

## 三、API 接口详情

### 1. 健康检查

**接口**：`GET /health`

**用途**：检查服务是否正常运行，获取缓存状态。

**响应示例**：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "status": "running",
    "cache_count": 553,
    "last_fetch": "2026-04-24T12:00:00",
    "refresh_interval_sec": 300
  }
}
```

---

### 2. 获取基金列表 ⭐（主要接口）

**接口**：`GET /api/funds`

**参数**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | number | 否 | 页码，默认1 |
| page_size | number | 否 | 每页条数，默认50 |

**用途**：获取所有LOF基金数据，用于列表展示。

**响应示例**：
```json
{
  "code": 0,
  "message": "success",
  "data": [
    {
      "code": "161725",
      "name": "招商中证白酒指数(LOF)",
      "price": 1.234,
      "nav": 1.189,
      "premium_rate": 3.78,
      "premium_status": "溢价",
      "change_pct": 2.45,
      "amount_w": 12345.67,
      "nav_date": "2026-04-23",
      "is_formal_nav": true
    }
  ]
}
```

**字段说明**：
| 字段 | 类型 | 说明 |
|------|------|------|
| code | string | 基金代码 |
| name | string | 基金名称 |
| price | number | 实时交易价格（元） |
| nav | number | 单位净值 |
| premium_rate | number | 溢价率（%），正数=溢价，负数=折价 |
| premium_status | string | 溢价状态："溢价"/"折价"/"平价" |
| change_pct | number | 涨跌幅（%） |
| amount_w | number | 成交额（万元） |
| nav_date | string | 净值日期 |
| is_formal_nav | boolean | 是否为正式净值（true=收盘后净值，false=盘中估算） |

## 四、数据处理建议

### 1. 异常数据过滤

溢价率超过 ±30% 的数据通常为异常（停牌基金价格固定为1.0），建议前端过滤：

```javascript
function filterValidFunds(funds) {
  return funds.filter(fund => {
    // 过滤无效溢价率
    if (fund.premium_rate === null || fund.premium_rate === undefined) {
      return false;
    }
    // 过滤异常溢价率（>50% 或 <-30%）
    if (fund.premium_rate > 50 || fund.premium_rate < -30) {
      return false;
    }
    return true;
  });
}
```

### 2. 溢价率颜色

```javascript
function getPremiumColor(rate) {
  if (rate > 0) return '#e74c3c';  // 红色 - 溢价
  if (rate < 0) return '#27ae60';  // 绿色 - 折价
  return '#333333';                 // 黑色 - 平价
}
```

### 3. 成交额格式化

```javascript
function formatAmount(amountW) {
  if (amountW >= 10000) {
    return (amountW / 10000).toFixed(2) + '亿';
  }
  return amountW.toFixed(2) + '万';
}
```

## 五、小程序开发建议

### 1. 数据缓存

后端每5分钟自动刷新数据，建议小程序：
- 启动时请求一次 `/api/funds`
- 本地缓存数据
- 手动刷新或定时刷新（建议≥60秒间隔）

### 2. 搜索实现

后端 `/api/funds` 接口支持分页，不支持搜索。小程序需自行实现前端过滤：

```javascript
function searchFunds(funds, keyword) {
  const kw = keyword.toLowerCase();
  return funds.filter(fund => 
    fund.code.toLowerCase().includes(kw) ||
    fund.name.toLowerCase().includes(kw)
  );
}
```

### 3. 排序实现

支持按以下字段排序：
- `premium_rate` - 溢价率
- `price` - 现价
- `nav` - 净值
- `change_pct` - 涨跌幅
- `amount_w` - 成交额

```javascript
function sortFunds(funds, field, order = 'desc') {
  return [...funds].sort((a, b) => {
    const valA = a[field] ?? 0;
    const valB = b[field] ?? 0;
    return order === 'asc' ? valA - valB : valB - valA;
  });
}
```

### 4. 错误处理

```javascript
async function fetchFunds() {
  try {
    const response = await wx.request({
      url: 'http://localhost:5000/api/funds?page=1&page_size=600',
      method: 'GET'
    });
    
    if (response.data.code === 0) {
      return response.data.data;
    } else {
      console.error('API错误:', response.data.message);
      return [];
    }
  } catch (error) {
    console.error('请求失败:', error);
    return [];
  }
}
```

## 六、完整示例

### 微信小程序请求示例

```javascript
// pages/funds/funds.js
Page({
  data: {
    funds: [],
    loading: true
  },

  onLoad() {
    this.fetchFunds();
  },

  async fetchFunds() {
    this.setData({ loading: true });
    
    try {
      const res = await wx.request({
        url: 'http://localhost:5000/api/funds',
        data: { page: 1, page_size: 600 }
      });
      
      if (res.data.code === 0) {
        // 过滤无效数据
        const validFunds = this.filterValidFunds(res.data.data);
        this.setData({ funds: validFunds });
      }
    } catch (e) {
      wx.showToast({ title: '加载失败', icon: 'none' });
    } finally {
      this.setData({ loading: false });
    }
  },

  filterValidFunds(funds) {
    return funds.filter(f => 
      f.premium_rate !== null && 
      f.premium_rate > -30 && 
      f.premium_rate < 50
    );
  },

  onSearch(e) {
    const keyword = e.detail.value;
    // 实现搜索逻辑
  }
});
```

## 七、注意事项

1. **跨域问题**：小程序请求需使用真实服务器地址，不能用 `localhost`
2. **域名白名单**：需将服务器地址加入小程序的 `request 合法域名`
3. **刷新频率**：建议≥60秒，避免频繁请求导致后端压力
4. **数据时效**：后端每5分钟刷新一次，盘后数据约15:00-16:00更新
