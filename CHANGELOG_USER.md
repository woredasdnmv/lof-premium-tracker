# 金快查 用户更新日志

## v1.4.0 (2026-05-20)

### 新增功能

- **自定义表头系统**：点击齿轮按钮打开表头管理弹窗，支持拖拽排序、勾选显隐、垃圾桶移除，偏好自动保存到本地
- **表头库**：将所有列分为「已显示」和「已隐藏」两个面板，一键勾选批量管理
- **表头存档**：可将当前表头配置保存为存档（如"套利专用"、"基础浏览"），随时切换不同视图方案
- **表头翻页**：当可见列总宽度超过表格时，表头行左右出现翻页箭头，点击即可翻页查看后续列
- **代码/名称列冻结**：表格左右滚动时，代码和名称列始终固定在左侧，方便对照

### 体验优化

- 搜索栏整体可点击区域扩大，点击空白处即可聚焦输入框
- 搜索栏宽度缩短，工具栏时间戳居中对齐
- 分页提示「显示 X 条，共 Y 条」靠右对齐
- 默认表头调整为 12 列：代码、名称、现价、净值、溢价率、三日均溢、成交额、预计收益率、预计收益额、停牌状态、申购状态、净值时间
- 表头管理弹窗采用 Modal 居中展示，与设置弹窗风格统一
- 拖拽排序动画采用 SortableJS，PC 和移动端均支持流畅拖拽

- **缓存状态可视**：搜索框占位符直接显示"(缓存x只 共x只) 代码/名称"，数据覆盖一目了然

### 体验优化

- 搜索栏同行右侧新增时间戳显示（格式：05/19 11:54 · 5分钟刷新），刷新状态一目了然
- 移除导航栏下方四格状态栏，界面更清爽
- 表头与数据列全部居中对齐，12 列宽度统一固定比例
- 溢价/折价排行榜跟随用户筛选设置实时联动

### 数据与性能

- **数据源全面升级**：弃用 AkShare，切换至东方财富（主源）+ 天天基金 / 腾讯 / 新浪 / 网易降级副源
- SSE 基金成交额数据修正：从流通市值切换为实际成交额，准确率大幅提升
- 净值优先级优化：盘后优先展示正式净值，盘中使用实时估算净值
- 热门基金图表预渲染缓存，打开即显无需等待

### 技术修复

- 修复排序字段为 None 时页面崩溃的问题

## v1.2.0 (2026-05-17)

### 新增功能

- **一年净值回填**：全市场 538 只 LOF 基金近一年日线净值数据全部补齐（72,567 行），曲线图场外净值不再断崖
- **筛选联动排行榜**：溢价/折价排行榜跟随用户筛选设置（停牌/暂停申购开关）实时更新
- **欢迎弹窗**：首次访问显示欢迎页面，包含使用协议确认、功能简介、QQ 交流群入口
- **隐私与协议页面**：新增隐私政策和使用协议页面
- **全屏图表（移动端）**：手机端曲线图支持全屏横屏查看，自动旋转屏幕方向

### 体验优化

- 默认隐藏停牌基金和暂停申购基金，用户可在设置中手动开启
- 预计收益为负时统一显示为"不建议交易"，避免误导
- 暂停申购基金的申购限额显示为"暂停申购"，预计收益自动归零
- 收益率计算中申购限额参与最大投入量计算，不再超额申购
- 收益构成明细弹窗支持停售基金的完整展示
- PC 端默认每页 20 条，翻页更轻松
- 成交额默认筛选门槛从 0 调整为 100 万
- 欢迎弹窗、设置弹窗、详情弹窗滚动条统一风格
- 页面移除所有 Emoji 图标，适配各平台字体渲染
- 所有预计收益处添加免责声明

### 数据与性能

- K 线数据源扩展至 9 个 API（东方财富 / 新浪 / 网易 / 腾讯 / Baostock / OpenBB / TuShare / AkShare / 管理器自动编排）
- 数据库日线覆盖 538 只 LOF 基金，91,697 行完整行情数据
- AkShare 数据源增加熔断保护（连续 3 次故障自动切换到备用源，5 分钟冷却）
- 数据库触发器阻止非交易日数据写入，避免周末数据错位
- 后端数据抓取默认返回全部基金（含停牌、停购），由前端按用户设置筛选
- 用户筛选设置持久化到浏览器 localStorage，跨会话复用
- 后端滚动更新前端 JS/CSS 版本号，确保用户加载最新代码

### 技术修复

- 修复 Chart.js CDN 地址返回 404 导致曲线图完全无法显示
- 修复暂停申购开关完全失效（后端 API 默认过滤 + 前端硬过滤双重拦截）
- 修复利润 ≤0 时收益构成字段缺失导致页面崩溃
- 修复前端 localStorage 被浏览器阻止时构造函数抛异常
- 隐私页面邮箱替换为 GitHub Issues 联系方式
- 代码仓库中所有硬编码数据库密码已脱敏并轮换

## v1.1.0 (2026-05-16)

### 新增功能

- **历史走势图**：基金详情页新增 365 天日线走势图，支持七日/一月/三月/六月/一年切换
- **溢价率曲线**：新增溢价率独立曲线图，红色为正溢价、绿色为折价，Y 轴颜色同步
- **套利模拟**：鼠标悬停走势图任意日期，自动计算该日作为套利起点的预计收益（溢价套利 T 日净值申购→T+2 卖出 / 折价套利 T 日买入→T+1 赎回，取收益更高者）
- **申购限额**：基金详情页显示申购限额，支持"不限额"标识，停售基金自动隐藏
- **曲线指标切换**：左侧下拉菜单切换"场内价格+场外净值"或"溢价率"图表
- **深色模式**：全局适配深色/浅色模式，滚动条、弹窗、图表坐标轴同步切换

### 体验优化

- 图表加载不再突然弹出，先显示空坐标系再填充数据
- 收益构成明细改为浮动弹窗，不挤占页面空间
- 套利提示改为软悬浮窗，悬停时出现在图表上方不遮盖曲线
- 基金详情弹窗改为上下对称居中，视觉更平衡
- 代码/名称旁标注"(点击可复制)"
- PC 端表格列宽优化，长名称不再换行，悬停显示完整名称
- 自定义滚动条样式，与页面主题一致

### 数据与性能

- 日线数据覆盖 396 只活跃 LOF 基金，近一年历史行情
- 实时净值自动区分盘中估算与盘后正式净值，溢价率计算更准确
- 热门基金图表预渲染缓存，打开即显无需等待
- 数据源智能切换，主源故障自动降级备用源，恢复后自动切回
